## 3. 核心结论（直接写「结论/讨论」，每条都有数据支撑）

1. **PEFT 全面优于全量微调（核心论点）**：AdaptFormer、LoRA、LoRA+SSF、SSF、BitFit 共 **5 种方法的平均准确率都超过 Full FT（91.79%）**，而它们的可训练参数都 **< 1.5%**。即"用不到 1.5% 的参数，效果反而更好"。

2. **全量微调在中小数据上过拟合**：Full FT 仅排第 6。在 4 个数据集里它都不是最好，在 Flowers（最小，2040 张）上甚至是**最低（99.43%，全场垫底）**。说明 100% 参数 + 小数据 → 破坏预训练特征、过拟合。

3. **AdaptFormer 综合最佳（93.00%）**：在 DTD、Pets 上均第一。代价是参数最多（1.43%，仍远小于全量）。

4. **本文 LoRA+SSF 组合的价值**：平均 92.57% 排第 3，**超过 Full FT（+0.78%）**；尤其在**最大的 CIFAR-100 上取得全场最高 93.90%**。说明：① 组合两种互补机制（LoRA 改注意力低秩更新 + SSF 逐通道缩放平移）是有效的；② 数据规模越大，组合的优势越明显。这是本课题"自己的改进与见解"。

5. **极简方法也很能打**：BitFit（只训 0.18% 的偏置）和 SSF（0.15%）平均都 > 92%，超过 Full FT。说明大量微调参数并非必要。

6. **Linear Probe 最差（89.71%）**：冻结整个骨干、只训分类头，表达力不足，是合理的"下限基准"。**任何一种 PEFT 都明显优于它**，证明"让骨干轻量地适配下游"确有必要。

7. **数据集难度差异**：Flowers 对预训练 ViT 近乎饱和（各法 99.4–99.7%，区分度小）；DTD（纹理）最难、区分度最大（78.3%–83.5%，跨度 5%+）；CIFAR-100、Pets 居中。**结论在"较难/中等"数据集上更显著**。

## 4. 消融实验：LoRA 秩 r（写「消融分析」用）

在 DTD 上固定 LoRA、只改秩 r：

| LoRA 秩 r | 可训练参数 | Top-1 (%) |
|---|---|---|
| 1 | 73,007 | 80.48 |
| 2 | 109,871 | 80.53 |
| 4 | 183,599 | 81.60 |
| 8 | 331,055 | 81.49 |
| 16 | 625,967 | **82.66** |
| 32 | 1,215,791 | **82.87** |

**结论**：① 准确率随秩 r 增大总体上升（80.5%→82.9%），但参数量也随之线性增长；② r 从 1→4 提升明显，4→8 基本持平，16/32 再上一个台阶；③ 在 DTD 这类较难任务上，**更大的秩能带来更高精度**，说明任务越难、所需的低秩更新"容量"越大。对应图：`fig5_lora_rank_ablation.png`。

## 5. 效率分析（写「效率/开销对比」用）

各方法峰值显存（多数据集平均）：

| 方法 | 峰值显存(MB) | 说明 |
|---|---|---|
| Linear Probe | ~1,493 | 最省——骨干冻结，无需保存骨干的激活梯度 |
| BitFit | ~5,621 | |
| AdaptFormer | ~6,043 | |
| VPT | ~6,085 | |
| LoRA | ~6,302 | |
| SSF | ~7,853 | |
| Full FT | ~8,546 | 需为全部 86M 参数保存梯度+优化器状态 |
| LoRA+SSF | ~8,642 | |

**结论**：① Linear Probe 显存最低（冻结骨干→不存骨干激活梯度）；② Full FT 显存最高（全参数梯度+优化器状态）；③ 多数 PEFT 介于两者之间，比 Full FT 省 25%~80%。
> 注：训练时间方面，Linear Probe 最快（329 img/s）；其余方法因梯度都要回传到网络深层，单步耗时相近。`lora_ssf` 在 CIFAR 的某次运行遇到散热降频，时间为离群值，分析时以**显存**为准更可靠。对应图：`fig6_efficiency.png`。

## 7. 写报告时的章节建议（小论文：摘要-引言-方法-结果-结论）

- **摘要**：一句话——在 ImageNet-21k 预训练 ViT-B 上对比 8 种微调方法、4 个数据集，发现多种 PEFT 用 <1.5% 参数即超过全量微调，并提出 LoRA+SSF 组合在大数据上达到最优。
- **引言**：迁移学习背景 → 全量微调的三大痛点（贵、存储多、易过拟合）→ PEFT 动机。
- **相关工作**：4 篇指定文献（LoRA / BitFit / AdaptFormer / SSF）+ VPT、Linear Probe 基准。
- **方法**：统一框架"冻结骨干 + 插入少量可训练参数"，分别讲 7 种方法 + 本文 LoRA+SSF 组合（配 `fig3` 参数占比、可画结构示意）。
- **实验**：设置（第1节）→ 主结果（第2节表 + `fig1/fig7/fig2`）→ 收敛性（`fig4`）→ 消融（第4节 + `fig5`）→ 效率（第5节 + `fig6`）。
- **讨论/见解**：第3节 7 条结论，重点写"PEFT 超过全量微调"+"LoRA+SSF 在大数据最优"这两个自己的发现。
- **结论与展望**：总结 + 展望（更大模型、更多任务、自适应秩等）。
