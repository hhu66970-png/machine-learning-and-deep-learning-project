# 9. 可解释性分析

前面各章已从"结果"层面回答了不同微调策略孰优孰劣:在 ViT-B 上,AdaptFormer、本文 LoRA+SSF 与 LoRA 三种参数高效方法的 8 数据集平均准确率(95.54%、95.26%、95.25%)全部超过全量微调(94.93%),而 Linear Probing(87.06%)与 VPT(92.54%)明显偏弱。然而,"谁更好"只是表象,真正具有学术价值的问题是**"为什么"**:为何冻结 99% 以上参数、只训练不到 0.5% 的旁路或缩放参数,反而能在大多数下游任务上击败把全部 86M 参数都拿来重训的全量微调?为何同属 PEFT,VPT 却显著落后?为何 LoRA 的损失曲面极其平坦而全量微调却异常尖锐?

本章用三种互补的可解释性工具,从"特征表示—参数更新—损失几何"三个层面,对前述性能差异给出**机理层面的解释**。三种分析分别回答三个递进的问题:(1) 微调改变了网络内部哪些层的特征表示、改变了多少(9.1 节 CKA);(2) 可学习的更新量在网络深度上如何分布、适配主要发生在哪里(9.2 节逐层适配强度);(3) 收敛到的极小值在参数空间中是平坦还是尖锐、这与泛化有何关系(9.3 节损失地形)。最后 9.4 节把三条证据链编织在一起,给出对方法性能差异的统一解释。所有可解释性实验均在 Oxford-IIIT Pet(细粒度近域任务、ViT-B 骨干)上进行,以保证横向可比。

## 9.1 CKA 层级特征相似度

### 9.1.1 方法论

要判断"微调把网络内部改了多少",最直接的做法是逐层比较**适配前(预训练初始权重)**与**适配后(微调收敛权重)**在同一批输入上产生的特征表示是否一致。但 ViT 各层的特征维度高、且存在旋转/缩放等无关变换,简单的逐元素距离无法刻画"表示是否本质相同"。本文采用中心核对齐(Centered Kernel Alignment, CKA)作为相似度度量[18]。

线性 CKA 的核心思想是:对两组特征矩阵 X(适配前)与 Y(适配后),先做样本中心化得到 X̃、Ỹ,再用 Hilbert-Schmidt 独立性准则(HSIC)度量二者的统计依赖,公式为

CKA(X, Y) = ‖Ỹᵀ X̃‖²_F / ( ‖X̃ᵀ X̃‖_F · ‖Ỹᵀ Ỹ‖_F )

其取值落在 [0, 1]:CKA = 1 表示两组特征在一个正交线性变换的意义下完全等价(即"表示没变"),CKA 越低则说明该层的特征表示被微调改动得越多。CKA 对各向同性缩放与正交变换不变,因而能剔除"换了一组等价基"这类表面差异,只保留实质性的表示改变,非常适合用来比较"微调到底动了哪些层"。本文取 Pet 测试集前若干个 batch 的样本,在 ViT-B 的 12 个 Transformer Block 输出处(取 [CLS] token 表示)分别计算预训练特征与各方法适配后特征的线性 CKA,得到图 11 的逐层曲线。

![图11 CKA各层特征相似度:不同微调方法适配前后逐层特征相似度(Pets, ViT-B)](ef_cka.png)

### 9.1.2 关键发现:全量微调"伤及浅层"而 PEFT"只改中层"

图 11 呈现出极为清晰、且与性能高度吻合的三档分层结构,本文逐一剖析。

**第一档——全量微调:浅层即遭重创,自下而上全面重写。** 图中粉色曲线(Full FT)的形态与所有 PEFT 方法截然不同。它在**第 1 层就已跌至约 0.83**,是所有方法中浅层 CKA 最低者;随后缓慢回升(中层约 0.90–0.91),到第 12 层才回到约 0.96。这条曲线传递的信息是触目惊心的:全量微调把本应承载"边缘、纹理、颜色、局部形状"等**通用低层视觉特征**的浅层网络也大幅改写了。然而对于 Pet 这种与 ImageNet 域差距很小的细粒度任务,这些低层特征本身已经几乎完美,根本不需要重学。把它们"训坏"非但没有收益,反而带来两大代价:其一是**过拟合**——浅层被有限的下游样本牵着重新拟合,泛化能力下降;其二是**灾难性遗忘**——预训练阶段在 1400 万张图上习得的、可迁移的通用先验被部分抹除。这正是全量微调在近域细粒度任务(Pets 93.76%、Flowers 99.43%、DTD 81.54%)上反而不敌多种 PEFT 的根本表示学层面原因。

**第二档——主流 PEFT(LoRA / SSF / AdaptFormer / BitFit / 本文 LoRA+SSF):浅层近乎不动,仅中层适度下沉。** 这一组方法的 CKA 曲线形成一个共同的"浅深两端高、中间凹"的浴盆形:浅层(第 1–3 层)CKA 普遍维持在 0.98–1.0 的高位,几乎完整保留了预训练的低层特征;真正的适配集中在网络**中层(第 5–9 层)**,此处 CKA 适度下降到约 0.95–0.97。具体而言,SSF 与 LoRA 的中层谷底约 0.96,BitFit 略高(中层约 0.974,改动最温和),而本文 LoRA+SSF 的中层谷底最低(第 6 层约 0.946、第 7–9 层约 0.952–0.957),说明 LoRA 与 SSF 的串联使中层获得了比单一方法更强的适配自由度。这一形态恰恰是 PEFT "扬长避短"的可视化证据:**冻结骨干这一结构性约束,把不可信赖的浅层"锁死"在预训练状态,只允许任务相关的中层语义被有限改写**——既适配了新任务的判别语义,又守住了通用低层特征不被破坏,从而在表示层面同时规避了过拟合与遗忘。这解释了为何这些方法能用 < 1.4% 的参数取得优于全量微调的泛化。

**第三档——VPT:几乎不触碰骨干,弱在"心有余而力不足"。** 橙色曲线(VPT)在全部 12 层都贴近 0.99–1.0,是所有方法中最"保守"的——这完全符合 VPT 的机制:它只在输入序列前拼接少量可学习 prompt token,完全不修改骨干内部任何权重,自然不会改变骨干在原 patch token 上的特征通路。CKA 几乎恒为 1 既是 VPT 的优点(绝不破坏预训练特征,因此在近域任务上不会崩),也精准暴露了它的软肋:**当任务需要骨干特征发生实质性改变时(如远域的 SVHN、GTSRB),VPT 缺乏改写中层表示的能力**。这与第 5 章主结果的观测严丝合缝:VPT 在 SVHN(90.33%)、GTSRB(88.78%)上大幅落后于其他 PEFT,平均仅 92.54%。CKA 曲线为"VPT 偏弱"提供了最直观的机理解释——它的适配能力被自身机制锁死在了"骨干之外"。

综上,图 11 第一次把"PEFT 为何优于全量微调"这一核心论点落到了可测量的表示证据上:**性能不是由改动的多少决定,而是由改动的位置决定。** 全量微调改得太多太浅(伤及通用特征),VPT 改得太少太外(够不着核心语义),唯有主流 PEFT 把适配精准地施加在"该改的中层"上,因而在泛化上胜出。这是本章最重要的发现,也是后续两节分析的逻辑起点。

## 9.2 逐层适配强度

CKA 回答了"哪些层被改变",但它度量的是**结果**(特征变了多少);本节进一步追问**原因**:可学习参数自身的更新量在网络深度上如何分布?适配的"力气"主要花在哪些层?本文对四种具备显式可量化更新量的方法,逐层统计其学习到的更新强度——对 LoRA 与本文 LoRA+SSF,统计低秩增量 ΔW = (α/r)·B·A 的范数;对 AdaptFormer,统计其瓶颈 adapter 等效更新 ΔW 的范数;对 SSF,统计缩放与平移偏离恒等的程度 |γ−1| + |β| 的范数。结果汇于图 12。

![图12 逐层适配强度:各方法学习到的更新量按 Transformer 层的分布(Pets, ViT-B)](ef_layer_adapt.png)

图 12 揭示了两个层面的规律。

**其一,绝对量级:AdaptFormer 的更新远大于其余三者。** 绿色曲线(AdaptFormer)的更新范数在 2.8–5.0 区间,峰值出现在第 5 层(约 5.0),整体比 LoRA、SSF、LoRA+SSF(多在 1.0–2.6 区间)高出约一倍。这与参数预算直接对应:AdaptFormer 可训练参数高达 1.20M(1.376%),是 LoRA(0.30M)的约四倍,拥有更大的"调整带宽",因而能注入更强的逐层变换。这也从更新强度角度解释了 AdaptFormer 取得最高平均准确率(95.54%)的来源——它以更高的参数代价换取了更充分的适配。值得注意的是,AdaptFormer 的更新在**浅层(第 1 层约 2.8)就已显著非零并迅速攀升至中层峰值**,提示其较大的容量使其更倾向于较早介入特征变换;不过结合 9.1 节,如此强的更新仍被"并联瓶颈、残差注入"的结构约束控制在不破坏浅层 CKA 的范围内。

**其二,分布形态:LoRA 系方法的适配单调偏向中后层。** 橙色(LoRA)、蓝色(SSF)与红色(LoRA+SSF)三条曲线形态高度一致:**浅层(第 1–4 层)更新量最小(约 1.0–1.5),随深度递增,在第 9–10 层附近达到峰值(约 2.0–2.6)。** 这一"前轻后重"的分布与 9.1 节的 CKA 结论彼此印证——浅层更新小,故浅层 CKA 高(特征几乎不变);中后层更新大,故中层 CKA 下沉(特征被实质改写)。两种独立度量从"因(更新量)"与"果(特征改变)"两端给出一致结论,大大增强了结论的可信度。其背后的机理是合理的:在 ImageNet-21k 预训练的 ViT 中,浅层学习通用的低级视觉模式,跨任务高度可复用,本就无需大改;而中后层编码任务特异的高级语义,正是适配新任务(如区分 37 类宠物的细粒度判别)最需要发力之处。PEFT 方法在没有任何显式层级先验、仅靠梯度自动分配的情况下,自发地把"力气"集中投向了最该适配的中后层——这本身就是 PEFT 之所以高效的内在机制:**有限的参数预算被自动调度到了边际收益最高的位置。** 此外,SSF 的曲线最为平缓(约 1.6–2.2),反映其"逐通道缩放+平移"是一种均匀、温和的全层微调;而 LoRA 与 LoRA+SSF 的更新在第 4 层之后陡升,层间分化更明显,说明低秩旁路更善于"挑层重点突破"。

## 9.3 一维损失地形

前两节考察了表示与参数;本节转向**优化几何**,即各方法收敛到的极小值在参数空间中的"形状"。大量研究表明,**平坦的极小值通常对应更好的泛化**:当极小值平坦时,测试分布带来的参数/数据微小扰动只会引起损失的微小上升,模型对扰动鲁棒;反之,尖锐极小值对扰动极度敏感,往往是过拟合的征兆[19]。

本文采用 filter-normalized 一维损失地形可视化[19]:从收敛点 θ\* 出发,随机采样一个方向 d 并按"滤波器范数"逐层归一化(以消除不同层权重尺度对可视化的干扰,保证不同方法间的曲线可比),然后绘制扰动后损失相对最小值的增量 L(θ\* + α·d) − L(θ\*) 随扰动幅度 α ∈ [−1, 1] 的变化曲线。曲线越平坦,极小值越平坦、泛化预期越好。结果见图 13。

![图13 一维损失地形:沿随机方向扰动时的损失增量(Pets, ViT-B)](ef_loss_landscape.png)

图 13 呈现出鲜明的三层结构,且与泛化性能高度相关。

**最平坦档——LoRA / AdaptFormer / Linear:** 紫色(LoRA)、棕色(AdaptFormer)与蓝色(Linear)三条曲线在整个 α ∈ [−1, 1] 区间几乎贴近横轴,损失增量始终 < 1.2。其中 LoRA 与 AdaptFormer 几乎完全平坦,意味着它们落入了极其平坦的盆地。这与它们优异且稳定的泛化性能(LoRA 平均 95.25%、AdaptFormer 95.54%)高度一致:**低秩与瓶颈结构把可训练参数约束在一个紧致的低维子空间内,等价于强正则化,从而天然地导向平坦极小值。** Linear 虽然也平坦,但那是因为它仅训练分类头、可变自由度极低,平坦更多源自"几乎没动",与高容量方法的"平坦"不可同日而语——这一点在 9.3 节末尾的可比性讨论中将进一步澄清。

**中等档——SSF / 本文 LoRA+SSF:** 红色(SSF)与品红色(LoRA+SSF)曲线呈中等开口,α = ±1 时损失增量约 3–6。这说明逐通道缩放/平移(SSF)以及"低秩+缩放"的组合(LoRA+SSF)收敛到的极小值平坦度居中。本文 LoRA+SSF 比纯 LoRA 略尖锐,是一个值得诚实记录的现象:叠加 SSF 引入了遍布全网的逐通道仿射自由度,使可训练子空间维度上升、正则强度相对减弱,因而极小值平坦度略有牺牲——这与其平均准确率(95.26%)虽超 LoRA(95.25%)但优势微弱、且在个别近域任务(DTD 81.75% < LoRA 82.07%)上略逊的现象相互呼应。

**最尖锐档——BitFit / Full FT:** 橙色(BitFit)与灰色(Full FT)曲线开口最大,α = ±1 时损失增量飙升至约 8–9,是所有方法中极小值最尖锐的两个。Full FT 的尖锐与 9.1 节的 CKA 证据形成闭环:它在 86M 维的全参数空间中自由优化,极易钻入对下游小数据集过度拟合的尖锐盆地,对扰动极度敏感,泛化风险高——这正是它在多数近域任务上反不及 PEFT 的优化几何层面解释。BitFit 同样尖锐则提示一个微妙之处:仅训练 bias 虽参数少(0.129%),但 bias 遍布每一层、且缺乏低秩/瓶颈那样的子空间约束,优化方向较"散",反而落入较尖锐的极小值,这与其平均准确率(94.63%)在 PEFT 中偏低相互印证。

**对可比性局限的诚实讨论。** 必须强调,一维损失地形在**不同参数量的方法之间存在固有的可比性局限**,不能简单地"曲线越平就越好"地横向裁判。原因有三:其一,各方法的可训练维度差异巨大(Linear 仅 ~7.7k、Full FT 高达 86M),filter-normalized 虽校正了层间权重尺度,却无法消除**子空间维度本身**对损失敏感度的影响——低维方法天然更"平",这部分平坦来自自由度少而非泛化好。其二,随机方向 d 是在各自的可训练参数空间内采样的,LoRA 的方向落在低秩子空间、Full FT 的方向落在全参数空间,二者并非同一几何对象,严格地说是"不同空间里的切片",跨方法比较只能作定性参考而非定量结论。其三,一维切片只是高维曲面的一条随机割线,可能错过关键的尖锐方向。因此本文将图 13 的结论限定为:**在"高容量且容量相近"的方法之间(如 LoRA vs AdaptFormer vs Full FT),平坦度差异是有意义的泛化信号;而在容量悬殊的方法之间(如 Linear vs Full FT),平坦度对比应谨慎解读。** 即便如此,"高容量的 LoRA/AdaptFormer 平坦、同样高容量的 Full FT 尖锐"这一对照仍是强有力的——它在控制了"容量"这一混杂因素后,清楚表明 PEFT 的结构约束确实导向了更平坦、更可泛化的极小值。

## 9.4 综合:三种分析如何共同解释方法性能差异

将三种可解释性证据汇聚到一起,本文得到一条自洽且互相印证的因果链条,完整地解释了第 5–6 章观测到的性能格局。三种分析分别从"改了哪里、用了多大力气、落到什么样的极小值"三个角度切入,而它们指向同一个结论。

为便于对照,本文把三条证据按方法整理如下:

| 方法 | CKA 浅层/中层(9.1) | 适配分布(9.2) | 损失地形(9.3) | 平均准确率 |
|---|---|---|---|---|
| Full FT | 浅层重创(~0.83) | —(全参数) | 最尖锐 | 94.93% |
| AdaptFormer | 浅深高、中层~0.95 | 更新量最大、中层峰值 | 极平坦 | 95.54% |
| LoRA | 浅深高、中层~0.96 | 中后层递增 | 极平坦 | 95.25% |
| LoRA+SSF(本文) | 中层最低~0.946 | 中后层递增 | 中等 | 95.26% |
| SSF | 浅深高、中层~0.96 | 全层温和均匀 | 中等 | 94.84% |
| BitFit | 中层~0.974(最温和) | —(仅 bias) | 较尖锐 | 94.63% |
| VPT | 全层~1.0(几乎不动) | —(骨干外) | 较平坦 | 92.54% |
| Linear | —(骨干完全冻结) | —(无) | 平坦(低容量) | 87.06% |

**第一条因果链——全量微调为何"高代价、低回报"。** CKA 显示它从浅层就大幅改写特征(9.1),损失地形显示它落入最尖锐的极小值(9.3),二者共同指向同一病因:在没有任何结构约束的 86M 维空间里自由优化,既破坏了不该动的通用浅层特征(过拟合/遗忘),又收敛到对扰动敏感的尖锐盆地(泛化差)。两条独立证据交叉验证了它在近域任务上不敌 PEFT 的原因——并非"调得不够",而是"调得过度且无约束"。

**第二条因果链——主流 PEFT 为何"低代价、高回报"。** 逐层适配强度显示它们自动把更新集中到中后层、浅层基本不动(9.2);CKA 因此显示浅层 CKA 高、仅中层适度下沉(9.1);损失地形则显示低秩/瓶颈的子空间约束把它们导向平坦极小值(9.3)。三者环环相扣:**结构约束 → 更新被自动调度到该适配的中层 → 浅层通用特征得以保全 → 收敛到平坦、可泛化的极小值。** 这条链条正是 "用 < 1.4% 的参数超越全量微调" 的完整机理,也解释了 AdaptFormer/LoRA/本文 LoRA+SSF 位居榜首的原因——AdaptFormer 以更大的更新量(9.2)换取最高精度,LoRA 以最强的平坦性(9.3)换取最稳健的泛化,二者是同一原理下的不同取舍点。

**第三条因果链——VPT 与 Linear 为何偏弱。** 二者偏弱的根源不同但都能被证据精确定位:VPT 的 CKA 几乎恒为 1(9.1),说明它从不修改骨干内部表示,适配能力被锁在骨干之外,故面对远域任务时"够不着"核心语义;Linear 则连提示都不加,完全依赖冻结特征,其损失地形的平坦(9.3)只是低容量的副产物而非泛化优势,在域差距大的 SVHN(65.08%)上彻底暴露了冻结特征的不足。

**对本文改进 LoRA+SSF 的机理定位。** 三种分析也为本文方法提供了细致画像:它在 CKA 上拥有最低的中层谷底(适配最充分,9.1),在适配分布上与 LoRA 同样偏向中后层(9.2),但在损失地形上比纯 LoRA 略尖锐(SSF 引入的全网仿射自由度削弱了正则强度,9.3)。这一画像恰好解释了它"以 0.437% 参数取得 95.26% 平均精度、超越全量微调且逼近 AdaptFormer,但相对纯 LoRA 优势有限"的实验结果——增加的适配自由度带来了精度的微小提升,却以平坦度的小幅牺牲为代价,是一种"精度—正则"权衡的合理产物。

综上,本章通过 CKA、逐层适配强度与损失地形三种相互独立、相互印证的可解释性工具,把"PEFT 为何优于全量微调"这一全文核心论断,从纯粹的准确率数字落实到了**特征表示、参数更新与优化几何**三个可测量的机理层面。三条证据链共同指向同一结论:**PEFT 的优势不在于改得多,而在于改得准、改得有约束**——它通过冻结骨干这一结构先验,自动地把有限的适配能力精确投放到最该适配的中层语义,在保全通用浅层特征的同时收敛到平坦可泛化的极小值。这为参数高效微调"少即是多"的反直觉现象提供了坚实而统一的机理解释。
