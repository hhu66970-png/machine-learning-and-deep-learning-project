# 2. 相关工作

参数高效微调(Parameter-Efficient Fine-Tuning, PEFT)并非孤立产生的技术，而是迁移学习范式与大规模预训练模型相互推动的必然结果。本章按"问题来源—方法谱系—评测与分析—本文定位"的逻辑展开：先回顾视觉Transformer与迁移学习的背景(2.1)，再依据"可训练参数长什么样、插在哪里"这一核心维度，把现有PEFT方法划分为三大谱系——参数选择类(2.2)、插入式低秩/瓶颈类(2.3)、特征调制与提示类(2.4)，逐一概括其核心思想、优缺点并相互对照；随后梳理PEFT的评测基准与分析工具(2.5)；最后阐明本文相对现有工作的差异与贡献(2.6)。本文涉及的全部方法机制对比汇总于表1，各方法在ViT-B上的可训练参数量见表5，主结果见表4。

## 2.1 视觉Transformer与迁移学习

视觉Transformer(Vision Transformer, ViT)将图像切分为固定大小的patch，经线性投影展平为token序列，叠加位置编码后送入标准的多头自注意力(Multi-Head Self-Attention, MHSA)与多层感知机(MLP)堆叠的Transformer编码器[1]。与卷积网络依赖局部感受野与平移等变的强归纳偏置不同，ViT几乎不携带空间先验，因此在中小数据集上从零训练往往欠拟合；但当在超大规模数据(如ImageNet-21k的约1400万张图、2万类[14])上预训练后，ViT能够学到高度通用且可迁移的视觉表征，并在下游任务上反超同规模CNN。这一"大数据预训练—小数据迁移"的特性，正是PEFT得以成立的前提：ViT中间层已编码了边缘、纹理、部件等通用特征，面向新任务时无需从零学起。

迁移学习的两种朴素做法构成了本文实验的上下界。**全量微调(Full Fine-Tuning)**解冻全部参数继续训练，上限最高，但需更新ViT-B的约86M参数，显存开销大、每个新任务须存储一份完整模型(本文实测每任务存储达343.33 MB，见表9)，且在小数据上易过拟合并损坏预训练特征(灾难性遗忘)。**线性探测(Linear Probing)**则冻结整个骨干、仅训练新分类头，极省极快、绝不破坏预训练特征，但表达能力受限，精度往往明显偏低——本文实测其8数据集平均仅87.06%，在与ImageNet域差距最大的SVHN上骤降至65.08%(表4)。二者之间的巨大鸿沟(平均约7.9个百分点)定义了PEFT的优化空间：以接近Linear Probing的代价，逼近甚至超越Full FT的精度。随着模型规模持续扩张，这一目标的工程价值愈发凸显，PEFT遂成为视觉迁移学习的核心研究方向。

## 2.2 参数选择类PEFT

最直接的参数高效思路是不引入任何新结构，仅从原模型已有参数中"挑选"极少子集进行训练，其余冻结。Linear Probing可视为此类的极端形式——只选择分类头。**BitFit**[3]则更进一步，冻结所有权重矩阵W，仅训练网络中无处不在的偏置项b(以及分类头)。其直觉在于：对线性变换 y = Wx + b 而言，偏置如同每层的"微调旋钮"，仅平移激活的分布即可让模型整体偏向新任务的决策区域，而无须改动承载主要表征的权重。BitFit原为BERT类语言模型提出，迁移到ViT后同样表现稳健：本文中其可训练参数仅110,602个(占比0.129%，表5)，却取得94.63%的平均精度，超过Linear Probing达7.57个百分点，逼近Full FT。

参数选择类方法的优点是实现极简、零额外推理结构、不改变模型拓扑、与任意架构兼容。其局限在于：可训练参数的"形态"被原模型结构完全限定，缺乏针对任务的表达自由度——偏置只能平移而无法重塑特征间的耦合关系。因此在域差距较大的任务上其上限受限，本文观察到BitFit在SVHN(95.02%)、GTSRB(96.01%)等远域数据集上落后于插入式方法(表4)，印证了"只拧旋钮"难以应对需要重塑特征的迁移场景。

## 2.3 插入式PEFT

插入式方法在冻结的骨干中**新增**少量可训练模块，赋予比参数选择类更强的任务表达能力，是当前PEFT的主流谱系。

**Adapter**[7]最早由Houlsby等人提出，在每个Transformer block的注意力与MLP之后**串联**插入一个瓶颈结构：先降维(d→r)、经非线性、再升维(r→d)，并以残差连接保证初始近似恒等映射。串联结构虽参数极少，但因处于主干前向路径上而引入推理延迟。**AdaptFormer**[4]针对ViT改进为**并联**瓶颈：在MLP分支旁并行一条 down→GELU→up 的旁路并乘以缩放系数s，与原MLP输出相加。并联设计在保持表达力的同时减少了对主干信息流的干扰。本文实测AdaptFormer取得8数据集最高平均精度95.54%(表4)，但其代价是相对较多的可训练参数(1,197,334个，占比1.376%)与最高的每任务存储(4.90 MB，表9)，体现了"以参数换精度"的取向。

**LoRA(低秩适配)**[2]从另一角度切入：全量微调对权重W学习的更新量ΔW，其内在秩往往很低，因而可用两个瘦长矩阵之积近似表示，前向计算为

h = W₀x + (α/r)·B·A·x

其中W₀为冻结的预训练权重，B为d×r、A为r×k矩阵，秩 r≪d，α为缩放系数。LoRA通常仅作用于注意力的Q、V投影，训练参数量从d×k降至r×(d+k)。其最大优势是**推理时可将B·A合并回W₀，实现零额外延迟与零结构改动**，这是相对串联Adapter的关键改进。本文中LoRA以302,602个可训练参数(0.351%)取得95.25%平均精度(表4)，并位列超越Full FT的三种方法之一；秩r的消融(表8、图10)进一步表明精度随r增大而提升，r=32时DTD达82.87%，揭示了表达力与参数量之间可调的权衡。

三类插入式方法相互对照：Adapter串联表达力强但有延迟；AdaptFormer并联精度最高但参数偏多；LoRA以低秩旁路兼顾精度与零延迟，参数—精度甜区表现突出。三者共同说明——为冻结骨干增设少量可学习的"特征再加工"通道，是逼近全量微调的有效途径。

## 2.4 特征调制与提示

第三类方法既不挑选原参数、也不插入瓶颈网络，而是通过**调制中间特征**或**注入提示输入**来引导冻结模型。

**SSF(Scale & Shift Features)**[5]在网络若干操作之后对特征施加逐通道的可学习缩放与平移：

y = γ⊙x + β

其中γ、β为可学习向量，⊙为逐元素相乘。直觉上，预训练特征已足够优良，只需对每个通道做类似"调亮度、调对比度"的仿射微调即可适配新任务。SSF参数极少(本文81,418个，占比0.095%)，且因γ、β为线性变换，**推理时可被重参数化吸收进相邻层权重，实现零额外开销**——这一点与LoRA的可合并性异曲同工，但作用粒度为逐通道仿射而非低秩更新。本文实测SSF平均94.84%(表4)，以不足BitFit的参数量取得更高精度，验证了通道级仿射调制比单纯偏置平移更具表达力。

**VPT(Visual Prompt Tuning)**[6]则完全不改动模型内部，而在输入patch token序列前**拼接N个可学习的"提示token"**，冻结骨干仅训练这些prompt(及分类头)，借自注意力让提示token与图像token交互、引导模型走向新任务。其参数量可极低(本文仅23,050个，占比0.027%，是除Linear外最省的方法)。然而VPT的表达完全依赖注意力对提示的间接利用，缺乏对特征的直接重塑能力，本文观察到其在远域任务上明显劣化(SVHN仅90.33%、GTSRB仅88.78%，表4)，平均精度92.54%，介于Linear与其余PEFT之间，体现了"暗号引导"范式上限受限的特性。

特征调制与提示类的共同特点是参数极少、对模型结构侵入性最小；区别在于SSF直接作用于特征通道、可重参数化、近域表现稳健，而VPT作用于输入空间、依赖注意力传播、在远域上较脆弱。

## 2.5 PEFT的评测基准与分析方法

随着PEFT方法激增，公平评测与机理分析成为关键。在**评测基准**层面，VTAB(Visual Task Adaptation Benchmark)[20]提出涵盖自然图像、专门领域(医学/卫星)与结构化任务三大类共19个数据集的标准协议，强调以统一的少样本与超参预算横向比较迁移方法，成为PEFT视觉评测的事实参照。本文的数据集设计正受其启发：兼顾细粒度近域(CIFAR-100、Flowers-102、Pets、DTD)与数字/卫星/交通标志等部分远域任务(CIFAR-10、SVHN、EuroSAT、GTSRB)，以考察方法的跨域鲁棒性(数据集统计见表2)。

在**分析方法**层面，理解PEFT"为何有效、改了什么"需要超越精度数字的工具。**CKA(Centered Kernel Alignment)**[18]通过中心化核对齐度量不同模型或层间表征的相似度，对正交变换与各向同性缩放不变，常用于比较微调前后各层特征的偏移程度，从而定位适配主要发生在浅层还是深层(本文据此绘制图11逐层CKA与图12逐层适配强度)。**损失地形(Loss Landscape)**可视化[19]则通过沿随机或特定方向对参数扰动、绘制损失曲面，刻画解的平坦度——平坦极小值通常对应更好的泛化与训练稳定性，可用于解释PEFT为何在小数据上不易过拟合(本文据此绘制图13一维损失地形)。这两类工具将"参数高效"的现象与表征几何、优化几何联系起来，是本文机理分析的方法学基础。

## 2.6 本文定位

综合上述，现有工作虽已提出丰富的PEFT方法，但多数论文聚焦于提出单一新方法并在各自选定的设置下报告优势，横向可比性不足。本文相对现有工作的差异与贡献体现在四点。其一，**统一公平比较**：本文在同一ImageNet-21k预训练骨干、同一数据管线与训练协议(超参见表3)下，统一实现并评测Linear、Full FT、BitFit、LoRA、AdaptFormer、SSF、VPT共7种代表性方法(连同本文提出的LoRA+SSF合计8种)，使表4中的精度、表5中的参数量与表9中的显存/存储开销直接可比，避免了跨论文设置差异带来的混淆。其二，**跨规模跨域系统研究**：本文同时在ViT-S、ViT-B两种骨干(表6、图6,并以ViT-L作为后续扩展)与8个覆盖近域到远域的数据集上评测，并补充5%—100%的数据效率曲线(表7、图8、图9)与LoRA秩消融(表8、图10)，系统刻画方法优势如何随模型规模、域差距与数据量而变化——这是单方法论文普遍缺失的维度。其三，**组合改进**：本文提出LoRA+SSF组合策略，将LoRA的低秩权重更新与SSF的逐通道仿射调制相结合，二者均可重参数化、互补而不冲突；该方法以0.437%的可训练参数取得95.26%平均精度(表4)，位列超越Full FT的三种方法之一，验证了不同PEFT机制叠加的可行性。其四，**机理分析**：本文不止步于精度排名，而是借助CKA特征相似度、逐层适配强度与损失地形，解释各方法"改动了表征的哪一部分、为何在不同域上表现分化"，力图给出超越榜单数字的独到见解。
