## 附录C 关键实现伪代码

以下伪代码节选并简化自 src/models/peft.py,完整保留各方法的核心数学结构与初始化策略。注释中给出对应公式,变量命名与源码一致。

LoRA(注入到注意力的融合 qkv 投影,仅作用于 q、v):

```python
class LoRAqkv(Module):
    # 包裹冻结的融合 qkv 线性层,对 q、v 各加一条低秩旁路
    # 数学: q = q + (alpha/r)·B_q·A_q·x ;  v = v + (alpha/r)·B_v·A_v·x
    def __init__(self, qkv, r=8, alpha=64):
        self.qkv = qkv                      # 冻结,不更新
        dim = qkv.in_features
        self.scaling = alpha / r            # 缩放系数
        self.A_q = Linear(dim, r, bias=False); self.B_q = Linear(r, dim, bias=False)
        self.A_v = Linear(dim, r, bias=False); self.B_v = Linear(r, dim, bias=False)
        kaiming_uniform_(self.A_q.weight); kaiming_uniform_(self.A_v.weight)
        zeros_(self.B_q.weight); zeros_(self.B_v.weight)   # B 初始化为 0 → 初始等价原模型

    def forward(self, x):
        qkv = self.qkv(x)                   # [..., 3*dim]
        q, k, v = split(qkv, dim)
        q = q + self.B_q(self.A_q(x)) * self.scaling
        v = v + self.B_v(self.A_v(x)) * self.scaling
        return cat([q, k, v], dim=-1)
```

SSF(逐通道缩放与平移,y = γ⊙x + β):

```python
class SSF(Module):
    # 数学: y = x * gamma + beta  (gamma、beta 为逐通道可学习向量)
    def __init__(self, dim):
        self.scale = Parameter(ones(dim)); self.shift = Parameter(zeros(dim))
        normal_(self.scale, mean=1.0, std=0.02)   # γ 初值近 1
        normal_(self.shift, std=0.02)             # β 初值近 0 → 初始近恒等
    def forward(self, x):
        return x * self.scale + self.shift

def inject_ssf(model):
    # 在每个 Block 的 norm1/norm2 之后、attn/mlp 之后各插一个 SSF
    for blk in model.blocks:
        blk.ssf_n1 = SSF(dim); blk.ssf_n2 = SSF(dim)
        blk.ssf_attn = SSF(dim); blk.ssf_mlp = SSF(dim)
        blk.forward = ssf_block_forward   # 改写前向: 调制 LN 输出与 attn/mlp 输出
```

AdaptFormer(并联瓶颈,挂在 MLP 分支旁):

```python
class Adapter(Module):
    # 数学: out = scale · up(GELU(down(x)))
    def __init__(self, dim, bottleneck=64, scale_init=0.1):
        self.down = Linear(dim, bottleneck); self.act = GELU(); self.up = Linear(bottleneck, dim)
        self.scale = Parameter(ones(1) * scale_init)   # 可学习标量缩放
        kaiming_uniform_(self.down.weight); zeros_(self.up.weight); zeros_(self.up.bias)  # 初始零输出

def adaptformer_block_forward(self, x):
    x = x + drop_path1(ls1(attn(norm1(x))))
    h = norm2(x)
    x = x + drop_path2(ls2(mlp(h))) + self.adapter(h)   # 并联: MLP 输出 + Adapter(h)
    return x
```

统一入口 apply_peft(冻结全部 → 注入模块 → 按名解冻 + 解冻头):

```python
UNFREEZE = {"linear_probe": [], "bitfit": [".bias"], "lora": ["lora_"],
            "adaptformer": ["adapter"], "ssf": ["ssf_"], "vpt": ["vpt_prompt"],
            "lora_ssf": ["lora_", "ssf_"]}

def apply_peft(model, method, cfg):
    if method == "full_ft":
        set_requires_grad(model, True); return model
    # 1) 注入方法特定模块
    if method in ("lora", "lora_ssf"): inject_lora(model, r=cfg.lora_r, alpha=cfg.lora_alpha)
    if method == "adaptformer":        inject_adapter(model, bottleneck=cfg.adapter_dim)
    if method in ("ssf", "lora_ssf"):  inject_ssf(model)
    if method == "vpt":                inject_vpt(model, num_prompts=cfg.vpt_prompts)
    # 2) 全部冻结,再按名字模式解冻 + 解冻分类头
    set_requires_grad(model, False)
    for name, p in model.named_parameters():
        if any(pat in name for pat in UNFREEZE[method]): p.requires_grad_(True)
    set_requires_grad(model.get_classifier(), True)
    return model
```

该统一框架的关键设计有三:其一,所有 PEFT 方法共用"先全冻结、再按名模式解冻 + 头"的单一接线逻辑,新增方法只需登记一条解冻模式;其二,LoRA 的 B 矩阵、AdaptFormer 的 up 投影、SSF 的 β 均初始化为零(或近恒等),保证训练初始时模型严格等价于预训练骨干,避免破坏已学特征;其三,LoRA+SSF 仅需把两条解冻模式取并集(`["lora_", "ssf_"]`),即可零成本组合两种正交的适配机制(低秩旁路负责注意力子空间更新,逐通道调制负责特征重标定),这正是本文改进方法实现上极为简洁的原因。

### D.3 复现要点与注意事项

第一,显存约束。本研究在 12 GB 笔记本 GPU 上完成全部实验,ViT-L 的全量微调批大小需降至 32,其余方法 64,其它骨干统一 128;若复现设备显存更小,可进一步减小批大小并相应线性缩放学习率。第二,结果聚合。run_campaign.py 将每次运行追加写入 outputs/results.csv(含 run_id、骨干、种子、训练比例等完整字段),analyze.py 读取该 CSV 做去重与按方法/数据集聚合后再绘图制表,因此中途崩溃重跑不会污染结果。第三,可复现性边界。即便固定随机种子 42,cuDNN 卷积与并行归约仍可能引入约 ±0.1~0.3 个百分点的浮动,故核心 4 数据集额外运行了种子 123、2024 以评估方差,正文与本附录所报数值均为对应运行的最优(best)Top-1。第四,中文图表渲染。analyze.py 已配置 Microsoft YaHei / SimHei 字体,在中文 Windows 环境下可直接渲染图题与图例,无需额外字体安装。
