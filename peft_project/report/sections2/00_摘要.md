# 摘要

随着视觉Transformer骨干规模膨胀至数千万乃至数亿参数,传统的全量微调暴露出算力与显存开销大、每任务需保存完整模型副本、小数据上易过拟合并损伤预训练特征三大痛点,难以支撑多任务与边缘部署。针对这一困境,本文构建了一个统一、公平的评测框架,在ImageNet-21k预训练的视觉Transformer上,以完全一致的骨干、数据增强、优化器(AdamW)与评测协议,系统研究8种微调策略——全量微调、线性探测、BitFit、LoRA、AdaptFormer、SSF、VPT,以及本文提出的组合改进LoRA+SSF。研究覆盖8个下游数据集(CIFAR-100、Flowers-102、Pets、DTD等近域细粒度任务与SVHN、EuroSAT、GTSRB等远域任务)、跨ViT-S/ViT-B规模,并系统开展5项消融(LoRA秩r、缩放系数α、AdaptFormer瓶颈维度、VPT提示数与学习率敏感性)、5%~100%数据效率曲线与多随机种子稳健性验证,辅以CKA层级特征相似度、一维损失地形、逐层适配强度三类可解释性分析,从精度、参数、显存、存储多维度综合刻画。关键结果显示:在ViT-B上,多种参数高效方法以不足1.5%的可训练参数,平均Top-1准确率超越全量微调的94.93%;其中AdaptFormer以95.54%居首,本文LoRA+SSF仅用0.437%参数取得95.26%、位列第2,LoRA(95.25%)紧随其后。本文进一步发现域差距是决定方法选择的关键因素,并通过CKA揭示参数高效方法几乎完整保留浅层与中层预训练特征、仅对高层施加任务特定修正,从而兼得抗遗忘与更平坦的损失地形。一项关键见解是:**全量微调对学习率高度敏感,学习率从1e-4增至3e-3时DTD准确率从81.9%崩塌至15.8%,而LoRA、SSF等参数高效方法在同一区间始终稳健且单调提升**;多种子实验亦显示PEFT跨种子标准差仅约0.1个百分点,印证其稳健性。本文据此形成一份可直接指导方法选型的完整图谱,对大模型时代的高效适配具有实践意义。

**关键词**:参数高效微调;视觉Transformer;LoRA;迁移学习;模型适配;可解释性
