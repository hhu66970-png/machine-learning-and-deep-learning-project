# 2. 研究方法

## 2.1 问题定义与统一框架

记在 ImageNet-21k[2] 上预训练得到的 ViT-Base/16[1] 骨干网络参数为 θ₀（共 85.8M 个参数），下游分类数据集为 D = {(xᵢ, yᵢ)}。微调的本质是在预训练参数附近寻找一个增量 Δθ，使下游任务损失最小：

min_{Δθ ∈ S} L(D; θ₀ + Δθ)

全量微调对应 S = ℝ^{|θ₀|}，即允许全部参数自由更新；而**参数高效微调（PEFT）的本质，是把增量 Δθ 约束在一个维度极低的参数子空间 S 中（dim(S) ≪ |θ₀|，本文各方法均不超过总参数的 1.5%）**。不同 PEFT 方法的差异，仅在于这个子空间"长什么样、插在哪"。

为明确"插补丁"的位置，先回顾 ViT-B/16 的结构：224×224 输入图像被切分为 196 个 16×16 的 patch，经线性投影得到 196 个维度 d = 768 的 token，并在序列首部拼接一个分类 token（cls token），共 197 个 token，随后送入 L = 12 个结构相同的 Transformer block。每个 block 采用 pre-norm 残差结构：

z′ = z + MSA(LN₁(z))

z″ = z′ + MLP(LN₂(z′))

其中 MSA 为 12 头自注意力，其 Q、K、V 投影在 timm[9] 实现中由一个融合的线性层 W_qkv ∈ ℝ^{3d×d} 一次性算出；MLP 为"Linear(d→4d) → GELU → Linear(4d→d)"的两层结构。由此可归纳出五类可插入可训练参数的位置：① 挑选原有参数的子集（BitFit）；② 给权重矩阵并联低秩旁路（LoRA）；③ 给 MLP 分支并联瓶颈子网络（AdaptFormer）；④ 在特征流上插入逐通道调制（SSF）；⑤ 在输入 token 序列中拼接提示向量（VPT）。

统一实验协议如下：所有方法均替换并**重新训练分类头** W_head ∈ ℝ^{C×d}（C 为类别数）；代码实现上先将全部参数置为 requires_grad=False，再按参数名模式（如 "lora_"、"ssf_"、".bias"）选择性解冻，保证各方法的可训练集合精确可控、参数量可精确统计。

## 2.2 基线方法：Full Fine-Tuning 与 Linear Probing

**Full Fine-Tuning（全量微调，上限基线）**对 Δθ 不加任何约束，更新全部 85.8M 个参数。其代价体现在三方面：① 显存——AdamW[10] 优化器需为每个参数维护一阶矩与二阶矩两份状态，加上梯度本身，训练态附加存储约为参数量的 3 倍，这是其峰值显存在所有方法中居于最高一档的根本原因；② 存储——每个下游任务都须保存一份完整模型（约 343MB），多任务部署成本线性增长；③ 泛化——在中小规模数据上，过大的自由度容易破坏预训练特征、引发过拟合（本文实验将证实这一点）。

**Linear Probing（线性探测，下限基线）**则完全冻结骨干，仅训练新的分类头，可训练参数只有 d×C + C 个（如 CIFAR-100 上为 76,900 个，四个数据集平均占比仅 0.064%）。由于骨干无任何梯度回传，无需保存骨干的激活与梯度，显存与时间开销最低；但骨干特征对下游域完全不适配，表达能力受限，精度构成所有方法的下界。PEFT 的目标正是：**以接近 Linear Probing 的代价，逼近乃至超过 Full FT 的精度**。

## 2.3 BitFit：只微调偏置项

网络中每个线性变换均形如

y = Wx + b

BitFit 冻结全部权重矩阵 W，**只训练所有偏置向量 b**（外加分类头）[4]。直觉上，偏置决定了每个神经元激活分布的"平移量"，只调整这些"旋钮"即可让冻结的特征通路偏向新任务。本实现中解冻的偏置包括每个 block 的 qkv 偏置（3d）、注意力输出投影偏置（d）、MLP 两层偏置（4d 与 d）以及两个 LayerNorm 的偏置（2d），合计每块 8,448 个，12 块共 101,376 个，再加 patch 嵌入与最终 LayerNorm 的偏置各 768 个，骨干侧共 102,912 个参数。计入随类别数变化的分类头后，实测可训练占比在 0.15%~0.21% 之间，四数据集平均 0.184%。BitFit 不引入任何新模块，推理结构与原模型完全一致，零额外开销。

## 2.4 LoRA：注意力投影的低秩适配

LoRA 的核心假设是：微调时权重的更新量 ΔW 具有很低的"本征秩"，因此可用两个瘦长矩阵的乘积参数化 ΔW = B·A，只训练 A、B 而冻结原权重[3]。对输入 x，适配后的输出为

h = W₀x + ΔWx = W₀x + (α/r)·B·A·x ,  A ∈ ℝ^{r×d}, B ∈ ℝ^{d×r}, r ≪ d

本实现细节如下（与源码逐一对应）：

- **注入位置**：仅对注意力的 Q、V 投影注入低秩旁路，K 保持不变。由于 timm 的 ViT 使用融合 qkv 线性层，本文以包装模块 LoRAqkv 实现：前向先用冻结的 W_qkv 算出 3d 维输出，按通道切分为 q = [0:d]、k = [d:2d]、v = [2d:3d] 三段，再分别用两组独立的低秩对 (A_q, B_q)、(A_v, B_v) 对 q、v 加上修正项 (α/r)·B·A·x，最后重新拼接，等价于对融合矩阵中 Q、V 两个子块做低秩更新。
- **初始化**：A 采用 Kaiming 均匀初始化（a=√5），**B 全零初始化**，故训练起点处 ΔW = B·A = 0，模型严格等价于预训练模型，避免随机扰动破坏预训练特征，训练稳定。
- **缩放因子**：修正项乘以 α/r 以解耦秩 r 与更新幅度，主实验取 r = 8、α = 8，即缩放为 1。
- **零推理延迟**：训练完成后可将 B·A 合并回原权重，W_q′ = W_q + (α/r)·B_q·A_q（V 同理），推理时无任何额外计算。

参数量为每个目标投影 r×(d_in + d_out) = 8×(768+768) = 12,288 个，乘以 2 个目标（Q、V）和 12 层，骨干侧共 294,912 个；计入分类头后实测平均占比 0.406%。

## 2.5 AdaptFormer：MLP 分支的并联瓶颈适配器

AdaptFormer 在每个 block 的 MLP 分支旁**并联**一个瓶颈结构的小网络[6]。记 h = LN₂(x)，适配后该子层输出为

x_out = x + MLP(h) + s · W_up · GELU(W_down · h)

即适配器与原 MLP 共享同一 LayerNorm 输出、并行计算后相加。本实现中瓶颈维度取 64（down: 768→64，up: 64→768，均带偏置），可学习的标量缩放 s 初始化为 0.1；与 LoRA 同理，**up 层权重与偏置全零初始化**，保证适配器初始输出为零、训练起点等价于预训练模型，down 层用 Kaiming 初始化。与经典串联 Adapter[5]（将瓶颈模块串接在子层之后、直接改变主通路）不同，并联设计让原始特征流原封不动通过，适配信号作为独立分支叠加，更适合保留 ViT 的预训练表征。每块参数为 (768×64+64) + (64×768+768) + 1 = 99,137 个，12 块共 1,189,644 个，实测平均占比 1.430%——在本文 PEFT 方法中最多，但仍远小于全量。其代价是 GELU 非线性使旁路无法合并，推理时存在少量额外计算。

## 2.6 SSF：特征的逐通道缩放与平移

SSF 假设预训练特征本身已足够好，下游适配只需对特征分布做"逐通道线性校准"[7]：

y = γ ⊙ x + β ,  γ, β ∈ ℝ^d

其中 ⊙ 为逐元素乘。本实现在每个 block 插入 **4 处** SSF 调制：LN₁ 之后（调制注意力输入）、注意力输出之后、LN₂ 之后（调制 MLP 输入）、MLP 输出之后，对应的 block 前向为

z′ = z + SSF_attn(MSA(SSF_n1(LN₁(z))))

z″ = z′ + SSF_mlp(MLP(SSF_n2(LN₂(z′))))

初始化取 γ ~ N(1, 0.02²)、β ~ N(0, 0.02²)，即近似恒等映射，同样保证温和的训练起点。每块参数 4×2×768 = 6,144 个，12 块共 73,728 个，实测平均占比 0.150%。SSF 的突出优点是**可重参数化**：由于 γ、β 是线性变换且紧跟在线性运算（LayerNorm 仿射或线性层输出 x = Wz + b₀）之后，推理时可被吸收进相邻权重：

γ ⊙ (Wz + b₀) + β = (diag(γ)·W)·z + (γ ⊙ b₀ + β)

吸收后网络结构与原模型完全一致，零额外推理开销。

## 2.7 VPT：可学习的视觉提示

VPT 不改动骨干内部，而是在输入 token 序列中拼接 N 个可学习的提示向量[8]。本实现取 N = 20，提示矩阵 P ∈ ℝ^{20×768} 以截断正态分布（std=0.02）初始化，并插入在 cls token 之后、patch token 之前：

Z₀ = [x_cls ; P ; x_patch]

序列长度由 197 增至 217。冻结的骨干通过每一层自注意力让 prompt 与图像 token 交互，从而"提示"模型走向新任务。本实现为**浅层 VPT（VPT-Shallow）**：可学习提示仅插入在输入层这一处，而非每层都注入独立提示（VPT-Deep），因此提示参数仅 20×768×1 层 = 15,360 个，计入分类头后平均占比 0.082%，是除 Linear Probing 外参数最少的方法。其代价在推理侧：序列加长约 10%，注意力计算量相应增加，且无法通过合并消除。

## 2.8 本文改进：LoRA+SSF 组合微调

**动机：两种机制在"作用对象"与"作用方式"上互补。** LoRA 对注意力 Q、V 投影施加方向性的低秩权重更新，改变的是 token 之间信息聚合的子空间——即"特征如何被混合"；SSF 对特征流做逐通道的缩放与平移，校准的是预训练域与下游域之间特征统计量（均值、幅度）的失配——即"特征如何分布"。前者作用于权重空间内部、容量随秩 r 可调但不触碰通道统计，后者作用于激活空间、擅长分布校准但秩为对角受限（diag(γ) 无法表达跨通道的方向性更新）。二者注入位置也不重叠（一个在 MSA 投影内部，一个在子层输入输出处），组合后有望同时覆盖"子空间重定向"与"分布校准"两类适配需求，取长补短。

**实现**：在同一骨干上同时执行 LoRA 注入（r=8、α=8，Q/V 投影，同 2.4 节）与 SSF 注入（每块 4 处，同 2.6 节），解冻模式为两者可训练参数的并集（"lora_" 与 "ssf_" 前缀）加分类头。两种模块都满足"零初始化/近恒等初始化"的性质（B 零初始化、γ≈1、β≈0），组合后的训练起点仍严格接近预训练模型，无需额外的稳定化技巧。骨干侧可训练参数为 294,912 + 73,728 = 368,640 个，计入分类头后实测平均占比仅 **0.491%**，不足全量的 1/200；且 LoRA 可合并、SSF 可吸收，组合方法推理时同样**零额外延迟**。

## 2.9 方法对比汇总

表1 从可训练位置、核心公式、推理开销与实测参数占比四个维度汇总全部 8 种方法（参数占比为四个数据集上含分类头的实测平均值，精确数字见第 3 节表4）。

**表1 各微调方法机制汇总**

| 方法 | 可训练位置 | 核心公式 / 机制 | 推理额外开销 | 可训练参数占比 |
|---|---|---|---|---|
| Full FT | 全部参数 | min L(D; θ) | 无 | 100% |
| Linear Probing | 仅分类头 | y = W_head·z | 无 | 0.064% |
| VPT[8] | 输入提示 token + 头 | Z₀ = [x_cls; P; x_patch] | 序列加长 20 token | 0.082% |
| SSF[7] | 每块 4 处 γ、β + 头 | y = γ ⊙ x + β | 无（可吸收重参数化） | 0.150% |
| BitFit[4] | 全部偏置 b + 头 | y = Wx + b，只训 b | 无 | 0.184% |
| LoRA[3] | Q/V 投影低秩旁路 + 头 | h = W₀x + (α/r)·B·A·x | 无（可合并） | 0.406% |
| LoRA+SSF（本文） | Q/V 低秩旁路 + 4 处 γ、β + 头 | 上两行机制叠加 | 无（均可合并/吸收） | 0.491% |
| AdaptFormer[6] | MLP 并联瓶颈 + 头 | x + MLP(h) + s·Up(GELU(Down(h))) | 少量旁路计算 | 1.430% |

图1 以对数坐标直观展示了各方法的可训练参数占比。可以看到，**全部方法的参数占比横跨三个以上数量级**：从 Linear Probing 的 0.064%、VPT 的 0.082%，到 SSF/BitFit 的 0.1%~0.2% 量级，再到 LoRA 系的 0.4%~0.5% 与 AdaptFormer 的 1.43%，最后跃升至 Full FT 的 100%。即便是参数最多的 AdaptFormer，与全量微调之间仍隔着近两个数量级；而本文组合方法 LoRA+SSF 在引入两种互补机制后，占比仍控制在 0.5% 以内。这一参数谱系为第 3 节的核心问题——"精度是否随可训练参数单调提升"——提供了天然的对照梯度：若答案为否，则说明微调收益取决于"参数插在哪、以何种机制起作用"，而非"训练了多少参数"。
