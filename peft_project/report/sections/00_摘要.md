# 摘要

全量微调预训练视觉Transformer存在算力与显存开销大、每个任务需保存完整模型副本、在中小规模数据上易过拟合三大痛点。针对这一问题,本文以ImageNet-21k预训练的ViT-Base/16(85.8M参数)为骨干,在"冻结骨干+少量可训练参数"的统一框架下,系统对比了Full Fine-Tuning、Linear Probing、BitFit、LoRA、AdaptFormer、SSF、VPT及本文提出的LoRA+SSF组合共8种微调策略,在CIFAR-100、DTD、Flowers-102、Oxford-IIIT Pet四个下游数据集上从准确率、可训练参数量、峰值显存、训练时间四个维度进行评测,并辅以LoRA秩(r=1~32)消融与收敛性分析。结果表明:AdaptFormer、LoRA、LoRA+SSF、SSF、BitFit共5种PEFT方法仅用不足1.5%的可训练参数,平均精度即全面超过全量微调(91.79%),其中AdaptFormer以93.00%居首;本文将注意力低秩更新与逐通道缩放平移两种互补机制叠加的LoRA+SSF组合取得92.57%的平均精度,并在规模最大的CIFAR-100上达到全场最高的93.90%,验证了互补机制组合在数据充足时的优势。上述结果证明,以极小参数代价完成下游适配不仅可行,而且往往优于全量微调;本文的统一评测框架与组合改进可为视觉模型参数高效微调的方法选型与超参数配置提供直接参考。

**关键词**:参数高效微调;视觉Transformer;LoRA;模型适配;迁移学习
