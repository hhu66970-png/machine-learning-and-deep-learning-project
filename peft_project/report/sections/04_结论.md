# 4. 结论

本文围绕"模型微调技术探索"这一课题,在 ImageNet-21k 预训练的 ViT-Base/16(85.8M 参数)[1][2]上构建了一个统一的参数高效微调实验框架:以"冻结预训练骨干、仅插入或选择极少量可训练参数"为公共范式,系统实现并对比了 Full Fine-Tuning、Linear Probing、BitFit[4]、LoRA[3]、AdaptFormer[6]、SSF[7]、VPT[8]以及本文提出的 LoRA+SSF 组合共 8 种微调策略,在 CIFAR-100[11]、DTD[12]、Flowers-102[13]、Oxford-IIIT Pet[14]四个下游数据集上完成了主实验、LoRA 秩消融与效率分析共 38 组实验,并从准确率、可训练参数量、峰值显存、训练时间四个维度进行了评测。

实验结果有力地回答了课题的核心问题。在平均 Top-1 精度上,AdaptFormer 以 **93.00%** 位列第一,LoRA 以 **92.60%** 次之,本文组合改进 LoRA+SSF 以 **92.57%** 排名第三,并在规模最大的 CIFAR-100 上取得全场最高的 **93.90%**,验证了"注意力低秩更新 + 逐通道缩放平移"两种互补机制组合的有效性;更重要的是,5 种主流 PEFT 方法均以不足 1.5% 的可训练参数全面超越了全量微调基线(平均精度 **91.79%**),"少量参数、更优效果"在全部 4 个数据集上得到一致验证。消融实验表明,LoRA 在 DTD 上的精度随秩 r 由 1 增至 32 总体上升(80.48%→82.87%),任务越难所需的低秩"容量"越大,而 r=4~8 已是性价比较高的选择;效率分析则显示 BitFit、AdaptFormer、LoRA 等方法的峰值显存比全量微调节省约 26%~34%,Linear Probing 因无需回传骨干梯度而最为轻量(节省 80% 以上)。

本课题的特点与意义在于:其一,所有方法共享同一骨干、数据管线与训练配方(AdamW[10]、余弦退火、混合精度),仅按方法调整学习率,保证了对比的公平性;其二,各方法均基于 timm[9]自行实现并与原论文机制逐一对照,完成了"实现—论文一致性"的验证;其三,实验结论可直接指导实践选型——一般任务首选 LoRA(r=8),追求极致精度选 AdaptFormer,参数预算极端受限时选 BitFit 或 SSF,下游数据量较大时可采用本文的 LoRA+SSF 组合。

本文工作仍存在若干不足:① 受时间与算力限制,每组实验仅以单一随机种子运行,未进行多种子重复与统计显著性检验,部分相近结果(如 LoRA 与 LoRA+SSF 的 0.03 个百分点差距)的可靠性有待确认;② CIFAR-100 仅训练 6 个 epoch,虽然预训练模型收敛很快,但部分方法可能尚未完全饱和;③ 实验仅覆盖 ViT-B/16 一种骨干与图像分类一种任务,结论的普适性需进一步验证;④ LoRA+SSF 在 DTD、Flowers-102 等小数据集上未超过单独 LoRA,组合策略的适用条件(数据规模、任务难度)尚缺乏深入刻画;⑤ 个别运行受笔记本散热降频影响,训练时间测量存在噪声,故效率分析以显存为主要依据。

未来工作可从以下方向展开:对全部实验引入多种子重复并报告置信区间;将对比框架扩展到 ViT-Large 等更大骨干以及自然语言处理任务(LoRA 本身即源于 NLP[3]);探索按层自适应分配秩的 LoRA 变体,把参数预算用在更关键的层上;对组合策略(如 LoRA+Adapter[5]、SSF+VPT)进行系统化搜索,厘清不同机制的互补规律;以及在极低数据量(few-shot)场景下考察各方法的鲁棒性,进一步逼近"以最小代价完成下游适配"的目标。
